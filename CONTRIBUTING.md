# Contributing to MindVault

Thanks for your interest in contributing! Here's how to get started.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/mindvault.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Set up your environment (see README.md Quick Start)

## Making Changes

- Keep changes focused — one feature or fix per pull request
- Follow the existing code style
- Test your changes locally before submitting

## Submitting a Pull Request

1. Push your branch: `git push origin feature/your-feature-name`
2. Open a Pull Request against `main`
3. Describe what you changed and why
4. Link any related issues

## Ideas for Contributions

- Semantic search using embeddings (replacing keyword matching)
- Persistent knowledge base (localStorage or SQLite backend)
- Better PDF extraction using pdf.js
- Dark mode toggle
- Export chat history
